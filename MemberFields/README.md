# Member Fields plugin

The Member fields plugin allows Administrators to add new fields to the Members profile.
It supports the following Field types
S = Simple Input
N = Number Input
T = Text Area
C = Check Box
D = Date
X = Time
Z = DateTime
O = Select Box
M = Multi-Select Box
H = Horizontal Radio Buttons
V = Vertical Radio Buttons
